Øvelser og opgaver:

    Øvelse 1: Hvis du ikke har processing i forvejen, så installer det og se at det virker ved at åbne det.
    Øvelse 2: Lav en profil på GitHub og se de to første videoer (1.1 og 1.2) i playlisten Git and GitHub for Poets. Brug videoerne som en tutorial, hvor du prøver at følge med Shiffman i dit eget repository på GitHub.
    Opgave 1: Hent GitHubs Desktop-klient her. Opret et nyt repository, som du løbende kan bruge til opgaver her på holdet. Brug i hvert fald dit fornavn i navngivningen af dit repository og tilføj din lærer som collaborator (brugernavn louissse). Brug Desktop-klienten til at hente dit repository ned lokalt. Opret en ny fil lokalt i dit repository og prøv at comitte og pushe den. Tjek på GitHub at filen dukker op i dit repository.
    Opgave 2: Gå sammen to og to. Tænk hver især på det sted i aller bedst kan lide at være. Beskriv stedet for din makker. Tag gerne noter. I skal nu bruge processing til at lave en skitse af jeres makkers sted. Brug Processing referencen til at finde flere former i kan bruge, link til referencen. Gem skitsen i jeres repository og husk at comitte og pushe.
